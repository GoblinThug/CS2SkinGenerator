# 🎯 CS2 Skin Generator **Автор:** [Goblin_Thug](https://github.com/GoblinThug)

Удобный инструмент для дизайнеров, разработчиков и создателей макетов рулеток CS2.

Приложение позволяет быстро находить скины, получать их изображения, названия, характеристики и генерировать готовые JSON-массивы для дальнейшего использования в интерфейсах кейсов, рулеток, инвентарей и других проектов.

---

## ✨ Возможности

### 🎨 Для дизайнеров

* Быстрый поиск нужных скинов
* Генерация карточек с реальными изображениями
* Настройка размеров изображений
* Удобное копирование изображений в буфер обмена
* Копирование названий оружия и скинов
* Просмотр редкостей и типов предметов

### ⚙️ Для разработчиков

* Генерация JSON-массивов
* Фильтрация по оружию
* Фильтрация по редкости
* Случайный StatTrak™
* Случайный Wear
* Быстрое копирование JSON
* Готовые данные для интеграции в рулетки и кейсы

### 🖥 Интерфейс

* Современный тёмный дизайн
* Кастомное окно приложения
* Встроенный DevTools
* Кастомные уведомления (Toast)
* Быстрая перезагрузка приложения
* Плавные анимации

## 📸 Что можно делать

### Генерировать случайные скины

Выберите:

* Редкость
* Оружие
* Количество предметов

После чего приложение сформирует набор случайных предметов.

---

### Получать изображения

Каждая карточка содержит реальное изображение скина.

Клик по изображению:

```text
Изображение → Буфер обмена
```

---

### Копировать названия

Клик по названию оружия:

```text
StatTrak™ AK-47
```

Клик по названию скина:

```text
Redline (Field-Tested)
```

---

### Генерировать JSON

Пример результата:

```json
[
  {
    "weapon": "AK-47",
    "skin": "Redline",
    "fullName": "StatTrak™ AK-47",
    "subtitle": "Redline (Field-Tested)",
    "rarity": "Classified",
    "stattrak": true,
    "wear": "Field-Tested",
    "image": "https://..."
  }
]
```

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/YOUR_USERNAME/cs2-skin-generator.git
```

### 2. Переход в папку проекта

```bash
cd cs2-skin-generator
```

### 3. Установка зависимостей

```bash
npm install
```

### 4. Запуск приложения

```bash
npm run dev
```

---

## 📦 Сборка приложения

После установки electron-builder:

```bash
npm install --save-dev electron-builder
```

Сборка Windows версии:

```bash
npx electron-builder
```

Готовые файлы появятся в папке:

```text
dist/
```
